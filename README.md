<h2 align="center" style="color:#3b4a2f;">Overview</h2>

<p align="center" style="max-width:700px;margin:auto;color:#555;">
Cohere is a unified platform that connects NGOs, donors, volunteers, and administrators 
into a single structured system. It transforms fragmented social initiatives into 
trackable, verifiable, and data-driven processes.
</p>
